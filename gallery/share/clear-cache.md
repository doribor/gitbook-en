# How to update a link preview in social media

There are situations when you share something on a social network or messenger, but it turns out that the preview picture is not the one you wanted or you need to change it. There is the way how to solve this problem.

## Clearing the Telegram cache <a href="#telegram" id="telegram"></a>

### WebpageBot <a href="#id-7538" id="id-7538"></a>

If you posted something in Telegram, you can clear the preview cache through a special bot: [@WebpageBot](https://web.telegram.org/#/im?p=@WebpageBot)

Follow the bot’s instructions. You will need to enter the links that need to be re-cached.

## Clearing the Facebook cache <a href="#facebook" id="facebook"></a>

[https://developers.facebook.com/tools/debug/](https://developers.facebook.com/tools/debug/)&#x20;

This is a tool for debugging shared links. You need insert the link you want to share and look at the result. If you need to reset the cache, then do it by clicking on the Scrape Again button.

## General way <a href="#general" id="general"></a>

You need to change the URL of the resource. For example, you can add something at the end, like ?v=1. You can write anything you want, the main thing is that the page continues to open with content.
