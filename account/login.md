# Access code for login

Logging into your account requires an **access code**, which is sent to your email. The code remains valid for 30 minutes.

#### Frequently Asked Questions

* **Why was the password removed?**\
  Previously, logging in with a password still required an additional access code as a mandatory security measure. Now, the process is simpler: the code remains, but the password is no longer needed.
* **What if I don't receive the code?**
  * Check your **Spam** and **Promotions** folders.
  * Ensure there is **enough storage space** in your email account or on your device for new messages.
* **Can I disable the access code?**\
  No, the access code is a mandatory security feature.
