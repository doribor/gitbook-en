# Sitemap.xml on your website

Sitemap is an xml file with links to the pages of your website, which helps search engines determine the addresses of pages on the website and the time of their last update.

File sitemap.xml it is generated automatically for all websites created on wfolio and is always kept up to date, being updated simultaneously with changes on your website.

Sitemap is always available for downloading via the link (where "yoursite" is your domain name):

```csharp
https://YOURWEBSITE/sitemap.xml
```
