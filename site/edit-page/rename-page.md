# How to rename a page

You can change the name of the page you created at any time.

To do this you need:

* Go to the <img src="https://files.carrotquest.io/knowledge-bases-images/articles/28780/28780-1642751290657-3gcr1heh.png" alt="" data-size="line"> [**Pages**](https://wfolio.com/my/site) section. Hover over the page you want to rename. Click the **ellipsis (â‹®)** at the end of the line and select **"Edit"**.

* You can change the name of the page in the **"Name"** item. **Save your changes.**
