# How to add a custom page

A custom page is a page with text, images, videos, forms, widgets, and more. It can be created from a ready-made template or you can make it from blocks by yourself. There are common examples of custom page:

* «About» page;
* Contact information with a feedback form;
* Services and prices.

Text pages with unique content have a beneficial effect on site promotion in search engines.



### **How to add a custom page**



How to add it step by step:&#x20;

* Go to the <img src="https://files.carrotquest.io/knowledge-bases-images/articles/28780/28780-1640171751454-pht7hv80.png" alt="" data-size="line"> [**Pages**](https://wfolio.com/my/site) section, click the **«Add to the Site»** button and select the **«Custom Page»** item.

<figure><img src="../../.gitbook/assets/image (320).png" alt=""><figcaption></figcaption></figure>

* Enter the name of the page (for example, «About me») and click **«Add»**.
