# How to make indents between blocks

If you need to increase or decrease the margins between blocks on a custom page, you can do this in two ways.

Let's look on a way to change the size of the indentation in the block settings:

* Open the custom page for editing. Click on the **Block Settings** (gear sign) and select the indent size in the **Top** or **Bottom** Indent item.

<figure><img src="../../.gitbook/assets/Frame 778.png" alt=""><figcaption></figcaption></figure>

You can also add an indent to the page in a separate block.

* Open the custom page for editing. Click on **"+"** to add a block. Go to the **"Separators"** section and select the **"Separator"** block.
