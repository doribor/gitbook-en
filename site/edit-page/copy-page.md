# How to copy a page

To create a copy of a page, go to the <img src="https://files.carrotquest.io/knowledge-bases-images/articles/28780/28780-1639927542715-nvh2hcb5.png" alt="" data-size="line"> [**Pages**](https://wfolio.com/my/site) section.&#x20;

Hover over the page in the general list of pages (or in the sidebar) that you want to copy. Click on the **ellipsis (⋮)** at the end of the line and select **«Copy the page»**.

{% hint style="info" %}
Only Custom pages can be copied.&#x20;
{% endhint %}
