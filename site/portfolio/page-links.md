# How to create subpages in a portfolio

<figure><img src="../../.gitbook/assets/subpages portfolio en.png" alt=""><figcaption></figcaption></figure>

**The subpages in the portfolio** is a page that contains galleries with albums. This method is suitable for you if you need to create a portfolio with three levels.

The variant has this structure:

> **Portfolio** (main custom page)
>
> * **Concerts** (album)\
>   — Muse (album)\
>   — Poets of the Fall (album)\
>   — Three Days Grace (album)\
>   — ...
> * **Sport** (album)\
>   — Marathon (album)\
>   — Olympic Games (album)\
>   — Figure skating (album)\
>   — ...
> * **Extreme** (album)\
>   — Mountain climbing (album)\
>   — Free diving (album)\
>   — Parachute jumping (album)\
>   — ...
> * — ...

How to create it:

* In the **Site** section click the **Add to site** button and add a gallery with albums. Hide it from the menu.

<figure><img src="../../.gitbook/assets/add gallery en.png" alt="" width="325"><figcaption></figcaption></figure>

* Add albums and upload photos.
* Copy the link to the created gallery.

<figure><img src="../../.gitbook/assets/copy page link en.png" alt=""><figcaption></figcaption></figure>

* Add a new custom page. Name it, for example, _Portfolio_.

<figure><img src="../../.gitbook/assets/add custom page en.png" alt="" width="325"><figcaption></figcaption></figure>

* Add blocks with images and text to the page.

<figure><img src="../../.gitbook/assets/Frame 776.png" alt=""><figcaption></figcaption></figure>
