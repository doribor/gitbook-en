# How to create a portfolio with tags

<figure><img src="../../.gitbook/assets/blog portfolio en.png" alt=""><figcaption></figcaption></figure>

You can create a portfolio page with filters that customers can use to search for the photo albums they need. This can be done using the "Blog" page.

How to do it:

* Click on the **Add to the site +** button and select **Blog**. Name the page and click the **Add** button.

<figure><img src="../../.gitbook/assets/Frame 889.png" alt=""><figcaption></figcaption></figure>

* Click on the **Add a post** button. Each post will be an album with photos. If necessary, allow photo downloads.
* Click on the <mark style="background-color:blue;">**+**</mark> button, add the **Gallery** block and upload your photos.

<figure><img src="../../.gitbook/assets/Frame 893.png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/Frame 894.png" alt=""><figcaption></figcaption></figure>

* To add categories, go to the **post settings** and in the **Categories** section, write a category for this post (album). Save changes.

<figure><img src="../../.gitbook/assets/post tags en.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
You can add a **maximum of 3 categories** for each post.
{% endhint %}

* Add more posts (albums) to the blog and add categories to them.

As the result, you have a portfolio with a filter by category and the possibility to download photos.
