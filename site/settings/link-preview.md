# Site link preview

The link cover is displayed next to a link to your site. This image is also called a _snippet_ or a _preview_.

The size of the cover depends on a social network. To make it suitable for Facebook, use a horizontal image with a size of at least **1200px**. In this case, your image will be reduced to the specified size and cropped in height to **630px** for Facebook.

A cover for social media will be updated in a few hours after loading, when the social media cache is updated.

{% hint style="warning" %}
The website preview image on social media **will update after a while**, once the cache is refreshed by the platform. If you don't want to wait, you can manually refresh the page cache.\
\
**You can do this according to our instructions:** [clear-cache.md](../pages/clear-cache.md "mention")
{% endhint %}

***

### Website social media cover

To upload the cover of the website (home page) for social media, go to **Settings → Site settings** and, at the very bottom of the page, upload an image in the **Social media cover** section.

This image will be shown for all page links of the website unless a specific preview is set for an individual page.
