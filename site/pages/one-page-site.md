# One page website with anchor links

You can make a one-page website with anchor links in the website menu, by clicking on which the page will scroll to the block you need.

* Click on the **Add to the site** button in the **Site** section and add a custom page.

<figure><img src="../../.gitbook/assets/add custom page en.png" alt="" width="325"><figcaption></figcaption></figure>

* Fill the page with your text and photos. After that you will need to copy the link to the block and add this link to the website menu so that when you click on it, the page scrolls to the block. To do this click on the **Settings** of the block and **copy the link** like`#section-5521834` .

<figure><img src="../../.gitbook/assets/copy anchor en.png" alt=""><figcaption></figcaption></figure>



* Click on the **Add to the site** button and select **Link**.

<figure><img src="../../.gitbook/assets/add link en.png" alt="" width="325"><figcaption></figcaption></figure>

* Name the link (this name will be displayed in the website menu) and in the **External link** section, paste the copied link. In the same way, copy links to other blocks on the page and add them to the website menu.

<figure><img src="../../.gitbook/assets/Group 230.png" alt="" width="539"><figcaption></figcaption></figure>

As a result, you will get a one-page website with anchor links in the menu, by clicking on which the page will scroll to the block you need.
