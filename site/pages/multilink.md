# Multilink page

You can only specify one clickable link in the profile description on Instagram and many other social networks. But one link is not enough if you need to specify links to the site, social networks, messengers and more.

You can do this with a simple custom page with a set of link buttons you need. In addition to buttons, you can also add text, slider, video and other blocks.

How to do it step by step:

* Go to the **Site** section, click the **Add to the site** button and select **Landing**. Name the page, for example, _Multilink_.
* You can change the name and page url in the **page settings**. Use short words as the page URL, for example, _links_, _menu_, or something similar.

<figure><img src="../../.gitbook/assets/link url.png" alt="" width="563"><figcaption></figcaption></figure>

* On a custom page, click the **+** button to add a new block. In the window that opens, choose the **Buttons** tab and select the block option with a button that suits you. Go to the button settings.
* In the window that opens, choose the action Open website page. Select the **homepage** of your site.

<figure><img src="../../.gitbook/assets/button settings site.png" alt="" width="563"><figcaption></figcaption></figure>



* Add buttons with your contact information. In the button settings, choose the click action:
  * **Go to social network**. Provide a link to Facebook, Instagram, YouTube, or Vimeo.
  * **Go to messenger**. Add a phone number for WhatsApp, Viber, or provide a link to Telegram.
  * **Make a call**. Add a contact phone number.

<figure><img src="../../.gitbook/assets/add instagram button.png" alt="" width="563"><figcaption><p>How to add Instagram button</p></figcaption></figure>

* To copy the link, click on **Actions** and select **Copy link**. You can post a link to this page on your social media profile.

{% hint style="info" %}
You can add any blocks to the page, but don't overload it with unnecessary information. The page should clearly describe who you are and contain convenient ways of contacting you.
{% endhint %}
