# Dropdown menu

The drop-down menu is used to combine several separate pages.&#x20;

How to do it step by step:&#x20;

* Go to the **Site** section, click on the **Add to site** button and select the **Drop-down menu** item.

<figure><img src="../../.gitbook/assets/add drop-down menu en.png" alt="" width="325"><figcaption></figcaption></figure>

* Name the drop-down menu (e.g., _My Works_) and click on the **Add**.
* The dropdown menu has been created. Now you can move existing pages or create new ones.
