# Page templates

Wfolio designers have prepared ready-made page templates so you can create your website quickly.\
All you need to do is replace the images in the templates and edit the text.

This article is about a **specific page template**. If you want to change the **site theme** (site styles, layout of elements), see the guide: [change-theme.md](../design/change-theme.md "mention")

### How to add a page template

* Add a **custom page** to your website.
* Click on the **Choose from pre-designed templates** and select the page template you like.

{% hint style="success" %}
Click on the **Preview** button to see how the template will look like with your website styles.
{% endhint %}

***

### How to change a page template

If you want to change a page’s template, delete the existing page with the current template:

* Select the **Actions** or (⋮) and select <mark style="color:red;">**Delete the page**</mark>.
* Add a new **custom page** and click on the **Choose from pre-designed templates**.
