# Custom page and blocks

A composite page is a custom page that can contain text, images, videos, forms, widgets, and more.\
You can create it from a ready-made template or build it yourself using blocks.

Examples of composite pages:

* _About me_ page.
* Contact information with a feedback form.
* Service pricing page.

Text-based pages with unique content have a positive effect on your websiteâ€™s search engine ranking.

### How to add a composite page

1. In the **Site** section, click on the **Add to website** and select **Custom page**.
2. Give the page a name and click **Add**.
3. Select **Choose from pre-designed templates** and start adding blocks to the page.
4. To add a block, click the **(+)** button and select the desired block.
