# Landing page

**Landing** is a separate page for advertising or selling your services, which is not displayed in the menu. On a page of this type there is no menu, logotype and footer at the bottom of the website.

To create a landing page, go to the **Site** section, click the **Add to site** button and select the Landing page.

<figure><img src="../../.gitbook/assets/add landing en.png" alt="" width="325"><figcaption></figcaption></figure>

Fill the page with information. To share a page, click the **Actions** button and **copy the link**.

### Converting a landing page to a custom page

If you need to enable the display of the landing page in the menu, then you can convert the landing page into a custom page. To do this, click on the **Actions** button and select **Convert to Page**.

{% hint style="info" %}
Any composite page can be converted into a Landing page. To do this, click on the **Actions** button and select **Convert to Landing**.
{% endhint %}

### Displaying the landing page in the menu

If you need to add a landing page on the website menu, then you can do it through the link. Click on the **Add to site** button and select **Link**. Name the menu item and select a landing page from the list of pages.

<figure><img src="../../.gitbook/assets/Group 229 (2).png" alt="" width="539"><figcaption><p>A link in the menu leading to a landing page with a sign up</p></figcaption></figure>

### What is the landing page for?

The landing page can serve as a promo page, a products sale page, a closed course for students, a multilink, etc.
