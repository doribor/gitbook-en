# Work with text

Text formatting is necessary to make the text structured and improve its perception by site visitors. Use different blocks and page templates. Don’t forget to use headings, lists, and text emphasis.

## Choosing the font, size and color of the text

You can choose one font for headings and one font for body text. Font combinations work well — a neat large serif font for a heading and a neat sans serif font for text, or vice versa. Use a well-readable, neat font on your website that is contrasting to the background. Remember that the main purpose of the font is to make the text on the site readable.

You can choose fonts and change the font size in the **Design** page in **Site fonts** section.

Сonfigure the settings, focusing on the preview of the text. After configuring, click the **«Save»** button at the bottom of the window.

## Typograph tool <a href="#typograph-tool" id="typograph-tool"></a>

The site uses the typograph tool by default. It helps adapt the text to the common typing rules. It automatically adjusts the quotation marks, dashes, spaces and punctuation.

To turn the typograph tool off go to the **Site settings** page and switch the **Typographic formatting** toggle in the **Other site settings** section.

## Text formatting

To format the text, use the built-in tools that are in the blue bar above the text.

![](../../.gitbook/assets/Screenshot_91.png)

By default, the input mode is set to straight text. For a beautiful and correct text formatting, we recommend using headings for the table of contents of blocks, as well as subheadings. Use of headings and a subheading to have a beneficial effect on the promotion of your site.

## New paragraph or line break?

There is always indentation before and after the text paragraph on a website, but sometimes you need to move the cursor to a new line without creating a new paragraph. To move the cursor to the next line without creating a new paragraph, press **Shift + Enter** (in Safari on Mac, **Ctrl + Enter**).

{% hint style="info" %}
**Enter** is a new indented paragraph. \
**Shift + Enter** (in Safari on Mac, **Ctrl + Enter**) is a new line without indents.
{% endhint %}

## How to make a beautiful list

It’s easy to make an even and beautiful list. To do this, highlight the text and select the **«List»** item in the text formatting panel:

![](../../.gitbook/assets/Screenshot_124.png)

The result will be an exactly formatted list, with the same indents and initial elements (bullets). If you already had your bullets at the beginning, then just delete them.
