# Creating and uploading a logotype

A logotype is an important element that will improve your recognizability among customers and site visitors. It’s easy enough to create a beautiful and stylish logo by yourself. There is one of the options for creating a logotype.

### How to make a logotype by yourself

{% hint style="info" %}
You can find a beautiful font in[ **Google Fonts**](https://fonts.google.com/). Examples of logos for inspiration can be found in [**Google Images**](https://www.google.ru/search?newwindow=1\&biw=2081\&bih=1073\&tbs=ic%3Atrans\&tbm=isch\&sa=1\&q=logo+photographer\&oq=logo+photographer\&gs_l=psy-ab.3..0j0i30k1l2j0i5i30k1.252165.252165.0.252739.1.1.0.0.0.0.68.68.1.1.0....0...1.1.64.psy-ab..0.1.67.hEjMqXmsCGA).
{% endhint %}

* Create a new file (press «Ctrl + N») with a size of at least **1600×1600 px** in any graphics editor (Photoshop or similar). Write your first and last name in a beautiful font («Text» Tool, «T» key). Then increase the font size so that the caption takes up almost the entire visible area.

<figure><img src="../../.gitbook/assets/image (345).png" alt=""><figcaption></figcaption></figure>

* Before saving, **delete** the «Background» layer to make the background of your logo transparent. Save the file in .PNG format with a transparent background.

<figure><img src="../../.gitbook/assets/image (245).png" alt=""><figcaption></figcaption></figure>

If you use a **light background** **color** of the website, then we recommend creating logos in two colors: **dark and white**.

### Why is it important to upload logos of two colors?

It is possible to set dark theme on different devices and browsers.

If you have **a light background color of the website**, and the website visitor turn on dark theme, then the background color on the visitor's screen will be changed to black, and the text color to white. Therefore, you will need a white logo.

If you have **a dark background color of the website**, then the enabled dark theme will not affect the background color of visitors. In this case, you only need to upload a white logo.

You can set the dark theme on the **Design** page in the **Site colors** section.

## How to upload a logotype

You can upload logos on the **Design** page in the **Logo** section. Here you can also change the height of the logo.
