# How to integrate email campaigns

You can install subscription forms on the website and later make your mailings based on the collected subscriber base. This can be implemented using an external service.

There are a lot of mailing services, we will tell you about [brevo.com](https://www.brevo.com/) as an example.

### **Brevo.com** ([brevo.com](https://www.brevo.com/))

**Limitations of the free version:** the number of subscribers is unlimited. Up to 300 emails per day.

**Pricing:** [https://brevo.com/pricing](https://www.brevo.com/pricing/)

What you will get:

* Automated mailing.
* Templates and customized forms for data collection.
* Integration with PrestaShop, Shopify, Magento, Google Analytics, and others.
* Simple interface.
* Detailed analytics.
* Mobile application.

### How to integrate

* Sign up for one the mail campaigns services.
* You will need an embed code that was copied it on the service's website to connect the mailing service.
* You can insert the embed code in the **Integration with services** section in the **head** text area.
* Or you can insert the code into the **HTML block** on the custom page of the website according to the instructions [html-code.md](html-code.md "mention").
