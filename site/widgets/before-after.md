# Before\&after widget

The **Before/after** widget is intended for visual comparison of two images within one block.

![](<../../.gitbook/assets/1 (36).png>)

A block with the widget can be added to any [custom page](../edit-page/add-custom-page.md). Click on the <mark style="background-color:blue;">**+**</mark> button and go to the **Widgets** tab in the modal window. Then select a **Before/after** block of any size you want.

![](<../../.gitbook/assets/Frame 787 (1).png>)
