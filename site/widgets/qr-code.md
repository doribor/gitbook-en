# How to create a QR code for a page

You can create QR codes for website pages. To do this, use any service that generates QR codes.

We will tell you about [qr-code-generator.com](https://www.qr-code-generator.com/) service as an example.

How to create a QR code for a project link:

* Open the project that you need to send to the client. Click on the **Share** button and **copy the link** to the project.

<figure><img src="../../.gitbook/assets/Frame 796.png" alt=""><figcaption></figcaption></figure>

* Go to [qr-code-generator.com](https://www.qr-code-generator.com/) service. Paste the link to the project in the text area and download the QR code.
