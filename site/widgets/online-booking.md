# How to connect an online booking system

The online booking system is a widget for your website, with the help of which customers and partners can book a date. Customers can see the current schedule, free hours and days.

We have reviewed several online booking systems.

* **Calendly** ([calendly.com](https://calendly.com/))\
  Free plan available.\
  Pricing: [https://calendly.com/pricing](https://calendly.com/pricing)\
  [How to get an embed code](https://help.calendly.com/hc/en-us/articles/223147027-Embed-options-overview)
* **Reservio** ([reservio.com](https://www.reservio.com/))\
  Free plan available (40 bookings).\
  Pricing: [https://www.reservio.com/pricing](https://www.reservio.com/pricing)\
  [How to get an embed code](https://help.reservio.com/en/articles/5663960-how-to-add-a-booking-form-to-your-website)
* **Simplybook.me** ([simplybook.me](https://simplybook.me/en/))\
  The free trial period is 14 days.\
  Pricing: [https://simplybook.me/pricing](https://simplybook.me/en/pricing)\
  [How to get an embed code](https://help.simplybook.me/index.php?title=Widgets_for_your_own_website)

### How to connect

* Sign up for one of the service.
* Fill in all the necessary information about you.
* Set up the system and copy the embed code. Check the service's instruction on how to connect the system.
* Insert the code into the **HTML block** on the custom page according to the instructions [html-code.md](html-code.md "mention").
