# How to add HTML code (widget)

To insert external services to your site, for example, online chat, order a call back, social media plugins and others that support integration using html-code, use a special block — **HTML-code.**

Sign up to a third-party service and receive an HTML code to place it on your website.

Then on the Custom page where you want to place the block, click on the link **+** (Add a block), select the **Widgets** tab in the opened window and scroll the window to the very bottom. Choose the size of the **HTML Code** block that you want.

![](<../../.gitbook/assets/Frame 787 (2).png>)

![](<../../.gitbook/assets/Frame 788.png>)

Now copy the HTML code of the external service and paste it into the block, save the changes.
