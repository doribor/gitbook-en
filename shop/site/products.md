# Offer products and services on the website

You can add an item that can be purchased immediately or ordered for purchase.



## How to add a product to the website

* [Create a product card in the **Shop**](../general/add-product.md).
* Go to or create a [custom page](../../site/edit-page/add-custom-page.md). Add a block with a button: press the <mark style="background-color:blue;">**+**</mark> button. Go to the **Buttons** tab and select a block.

<figure><img src="../../.gitbook/assets/8 (16).png" alt=""><figcaption></figcaption></figure>

* Go to the button settings.

<figure><img src="../../.gitbook/assets/10 (10).png" alt=""><figcaption></figcaption></figure>

* Name the button. For the **Click action**, select **Open product/service**. If necessary, enable the option to open products in a new tab.

<figure><img src="../../.gitbook/assets/add product card site_.png" alt="" width="563"><figcaption></figcaption></figure>

* Fill a page with the necessary information. Add a description of your product and save changes.

Clicking on the button will open an order form. Depending on the settings, a form opens in a modal window on a current page or in a new tab.
