# How to create a personal account of the client

You can make pages that will be available to a limited number of users (like personal account of users). The list of users who will have access to the page, that you can make by yourself. This type of access can be granted for a limited and unlimited period.

Access to the page is provided by sending unique one-time passwords to the mail of users.

### How does a one-time password work?

When opening a page with one-time password, the user needs to enter the email for access. The password is sent only to the email address that is in the customer database for this page.

<figure><img src="../../.gitbook/assets/Frame 826.png" alt=""><figcaption></figcaption></figure>

### How to add a page with one-time password

* Go to the page settings.

<figure><img src="../../.gitbook/assets/open page settings wfolio.com.png" alt=""><figcaption></figcaption></figure>

* Go to the **Access** tab and select **One-time password** type of access.

<figure><img src="../../.gitbook/assets/set one-time password en.png" alt="" width="563"><figcaption></figcaption></figure>

After that, you will have the **Clients** tab, where the database of clients who have been granted access to the page is stored. Here you can add new records and edit existing ones.

You can change the client's name, email, and also limit the period of access to the page in the client settings.

<figure><img src="../../.gitbook/assets/clients with access en.png" alt=""><figcaption></figcaption></figure>

### How to add access to the personal account after purchasing the product

* Go to the Shop to [add a product card](../general/add-product.md).
* Navigate to the **Actions after the order** tab.
* Enable redirect to a **Website page** protected with one-time passwords. If necessary, limit access period.

<figure><img src="../../.gitbook/assets/access after order en.png" alt="" width="563"><figcaption></figcaption></figure>
