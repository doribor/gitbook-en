# Currency

The shop's default currency is set to US dollars.

To change the currency, go to the **Shop settings** tab in the **Shop**. Select your preferred currency from the list. One site or drive is limited to a single currency.

<figure><img src="../../.gitbook/assets/store currency.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Keep in mind that selected currency may not be available for accepting payments online. [Payment systems](payment-systems.md) work with a certain set of currencies and vary by region.
{% endhint %}
